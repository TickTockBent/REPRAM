Quick Start

Run a Node

git clone https://github.com/ticktockbent/REPRAM.git
cd REPRAM
make build
./bin/node

Docker Deployment

docker-compose up -d
# 3-node cluster ready at localhost:8080-8082

Store Data (Raw API)

curl -X PUT "http://localhost:8080/data/mykey?ttl=300" \
  -d "sensitive data"
